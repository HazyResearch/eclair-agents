# eclair-agents

FMs for Workflow Automation

## Installation

```bash
git clone https://github.com/HazyResearch/eclair.git
cd eclair-agents/
conda create -n eclair_env python=3.10 -y
conda activate eclair_env
pip install -r requirements.txt
pip install -e .
```
